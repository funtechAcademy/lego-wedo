Introducción

En este proyecto el alumno debe construir la base de los dos siguientes montajes. Debe hacer una serie de retos iniciales para ver como se realiza el movimiento y como se traslada de un sitio a otro.

El primer montaje tiene las instrucciones para construir la parte de los engranajes. Este robot será la base para los dos siguientes robots.

Los dos robots siguientes no tienen instrucciones y solo disponen de 3 fotos. Se construyen sobre la base del montaje anterior.

Objetivos de la Actividad

El alumno debe comprender como funciona el mecanismo de piñón y cremallera.

Además veremos como se utiliza un trinquete para bloquear el retroceso.

Retos

Primera Fase

• Hacer que el robot mueva el mecanismo adelante y atrás.

• Hacer que espere al sonido de una palmada para salir y otro para entrar.

• Hacer que en cada estado haga un sonido.

Segunda Fase

Ejemplo de Programación

• Hacer que la oruga se mueva de un extremo a otro de la mesa.

• Repetir pero esperando a una palmada para iniciar la marcha.

Tercera Fase

Ejemplo de Programación

• Hacer que la mantis mueva las pinzas de atrás hacia adelante.

• Repetir usando el sensor de movimiento.
