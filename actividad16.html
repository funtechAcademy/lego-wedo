Vaivén

Introducción

En este proyecto el alumno debe construir la base del los dos siguientes montajes. El alumno debe realizar una serie de retos iniciales para ver como se realiza el movimiento y como se traslada de un sitio a otro.

Objetivos de la Actividad

Retos

Primera Fase

• Hacer que el robot mueva el apéndice de un lado a otro todo el tiempo.

• Hacer que el robot haga un sonido cada vez que toca un extremo al moverse.

Segunda Fase

Ejemplo de Programación

• Hacer que la compuerta se abra y se cierre utilizando un bucle.

• Hacer la secuencia de apertura de compuerta, esperar que el barco pase y cerrarse.

• Repetir anterior además haciendo sonidos.

Tercera Fase

Ejemplo de Programación

• Hacer que el pez mueva la cola cada vez que detecte el sonido de una palmada.

• Repetir anterior haciendo que haga un sonido cada vez que toca un extremo.
