Barrer

Introducción

En este proyecto el alumno debe construir la base del los dos siguientes montajes. Debe hacer una serie de retos iniciales para ver como se realiza el movimiento y como se traslada de un sitio a otro.

El primer montaje tiene las instrucciones para construir la parte de los engranajes. Este robot será la base para los dos siguientes robots.

Los dos robots siguientes no tienen instrucciones y solo disponen de 3 fotos. Se construyen sobre la base del montaje anterior.

Objetivos de la Actividad

El alumno debe aprender el funcionamiento de las dobles poleas y de como estas pueden compartir la fuente de movimiento, con la diferencia de girar en sentido contrario una respecto de la otra.

Retos

Primera Fase

• Hacer que giren las aspas.

• Cada vez que se de una palmada, las aspas deben cambiar de sentido.

• Dependiendo del sentido de giro, debe mostrar un color u otro.

Segunda Fase

Ejemplo de Programación

• Hacer que el barco sea capaz de recoger toda la basura marina.

• Hacer que inicie el movimiento cuando reciba el sonido de una palmada.

Tercera Fase

Ejemplo de Programación

• Hacer que la barredora lleve la basura a otro lugar.

• Repetir el ejercicio haciendo que antes detecte la basura con el sensor de proximidad.

• Repetir pero haciendo que espere al sonido de una palmada para iniciar.
