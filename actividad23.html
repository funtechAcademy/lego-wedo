Barrer

Introducción

En este proyecto el alumno debe construir la base del los dos siguientes montajes. El alumno debe realizar una serie de retos iniciales para ver como se realiza el movimiento y como se traslada de un sitio a otro.

Objetivos de la Actividad

Retos

Primera Fase

• Hacer que giren las aspas.

• Cada vez que se de una palmada, las aspas deben cambiar de sentido.

• Dependiendo del sentido de giro, debe mostrar un color u otro.

Segunda Fase

Ejemplo de Programación

• Hacer que el barco sea capaz de recoger toda la basura marina.

• Hacer que inicie el movimiento cuando reciba el sonido de una palmada.

Tercera Fase

Ejemplo de Programación

• Hacer que la barredora lleve la basura a otro lugar.

• Repetir el ejercicio haciendo que antes detecte la basura con el sensor de proximidad.

• Repetir pero haciendo que espere al sonido de una palmada para iniciar.
