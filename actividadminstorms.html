Introducción

Construiremos para las 4 actividades el mismo robot base. A través de la aplicación en el móvil o la tablet, utilizaremos el dispositivo como mando a distancia, para realizar las diferentes competiciones.

Robot Base

Construiremos el robot base mediante las siguientes instrucciones.

Control del Robot

Para poder controlar el robot, utilizaremos la aplicación LEGO MINDSTORMS COMMANDER.

Podemos descargar la aplicación en:

- Para Android:

   LEGO MINDSTORMS COMMANDER

- Para IOS:

   LEGO MINDSTORMS COMMANDER

A la hora de utilizarlo por primera vez, tenemos que configurar la pantalla de control.

Abrimos la aplicación. Nos aparecerá la pantalla principal con los distintos montajes.

Competiciones

Fútbol

• Hacer que el robot vaya de un extremo a otro de la mesa sin chocarse con el tope de la misma.

• Hacer que el robot vaya y vuelva de un extremo a otro de la mesa.

• Hacer que robot vaya a un extremo de la mesa, deteniéndose por el camino a la altura de un bloque propuesto por el profesor (pueden ser uno o mas bloques).

• Repetir el ejercicio anterior y además el robot debe regresar de un extremo de la mesa haciendo las paradas que hizo al ir.

• Repetir el ejercicio anterior haciendo que el robot haga sonidos en cada parada.

Milo con Sensor de Proximidad

En este reto, el alumno debe realizar el montaje anterior (Milo el Robot Explorador) y a continuación, deberá acoplar el sensor de proximidad.

Este sensor permite detectar un objeto que este próximo, además de mostrar la distancia a la que se encuentra.

El rango de detección máximo es de 10 cm.

Objetivos de la Actividad

En esta sección se introducirá a los estudiantes en el uso del sensor de movimiento para detectar la presencia de una nueva especie de planta.

El alumno debe aprender a usar el sensor de proximidad y entender el funcionamiento de éste, ademas de aprender a combinar las señales que devuelve con los bloques de programación.

Retos

• Hacer que el robot vaya de un extremo a otro de la mesa sin chocarse con el tope de la misma.

• Hacer que robot vaya de un extremo a otro de la mesa y se detenga cuando detecte el tope de la mesa con el sensor.

• Repetir el primer ejercicio pero haciendo que el robot detecte una señal al sensor para iniciar la marcha.

• Hacer que el robot vaya de un extremo a otro de la mesa, deteniéndose a la altura de bloques propuestos por el profesor, utilizando para ello el sensor de proximidad.

• Repetir el ejercicio anterior haciendo que el robot haga un sonido en cada parada.  

Milo con Sensor de Inclinación

El alumno deberá realizar el montaje de Milo básico para, a continuación, acoplarle el sensor de inclinación.

Este sensor detecta las diferentes posiciones que puede tener (inclinado a la derecha, a la izquierda, hacia delante, hacia atrás y sin inclinación).

Objetivos de la Actividad

En esta sección se introducirá a los estudiantes en el uso del sensor de inclinación, que ayudará a Milo a enviar un mensaje a la base.

El alumno debe aprender a usar el sensor de inclinación y entender el funcionamiento de éste. Además debe aprender a combinar las señales que devuelve con los bloques de programación.

Retos

• Hacer que el robot vaya de un extremo a otro de la mesa sin chocarse con el tope.

• Repetir el ejercicio anterior iniciando la marcha solo si con el sensor inclinado hacia adelante.

• Repetir ejercicio anterior haciendo que vuelva solo si el sensor esta inclinado hacia adelante.

• Repetir el ejercicio anterior haciendo que el robot emita un sonido cada vez que cambie de sentido.
