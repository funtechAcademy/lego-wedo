Velocidad

Introducción

En este proyecto, el alumno debe construir un coche de carreras.

Veremos como uniendo el motor a las ruedas mediante una goma, podemos transmitir el movimiento del motor a las ruedas.

Aprenderemos como poniendo la goma de un engranaje más pequeño a otro más grande y viceversa, podemos cambiar la velocidad de nuestro coche.

Además este coche tiene incorporado el sensor de infrarojos por lo que se debe hacer uso de él como en proyectos anteriores.

Objetivos de la Actividad

El objetivo de este proyecto es investigar los factores que pueden hacer que un coche vaya más rápido para ayudar a predecir el movimiento.

El alumno debe aprender a utilizar los principios de ejes grandes y pequeños y de como la combinación de estos afecta a la velocidad.

Por otro lado, seguiremos aprendiendo el manejo del sensor de infrarojos.

Retos

• Hacer que el robot vaya de un extremo a otro de la mesa sin chocar con el tope.

• Repetir el ejercicio anterior y además hacer que el coche vuelva.

• Repetir el ejercicio anterior haciendo que para no chocarse utilice el sensor de infrarojos para detectar el tope de la mesa.

• Repetir el ejercicio anterior pero además el robot debe esperar una señal en el sensor de infrarojos para iniciar la marcha.

• Opcionalmente podemos hacer que el robot se detenga durante su recorrido en marcas propuestas por el profesor.
